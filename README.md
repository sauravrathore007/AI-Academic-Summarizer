# AI-Based Text Summarization System for Academic Notes

This project is a Generative AI application designed to help students summarize long academic notes using Large Language Models (LLMs).

## 🚀 Key Features
- **Instant Summarization:** Converts long paragraphs into key points.
- **Academic Focused:** Optimized for research papers and study materials.
- **Simple UI:** Built with Streamlit for a smooth user experience.

## 🛠️ Technical Stack
- **Python:** Primary programming language.
- **Hugging Face:** Used the `DistilBART` model for high-quality summarization.
- **Streamlit:** Used for creating the Web-based Graphical User Interface.

## ⚙️ How to Run
1. Install requirements: `pip install streamlit transformers torch`
2. Run the app: `streamlit run app.py`
